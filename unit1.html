1.1 Python的特点和优势 ✨

Python是一种简单易学、功能强大的编程语言，具有以下特点和优势：

  • 语法简洁：Python的语法非常简洁明了，代码可读性高，学习曲线平缓，适合初学者。
  • 跨平台：Python可以在Windows、macOS、Linux等多种操作系统上运行。
  • 丰富的库：Python拥有大量的标准库和第三方库，可以轻松实现各种功能。
  • 用途广泛：Python可用于Web开发、数据分析、人工智能、自动化脚本等多个领域。
  • 社区活跃：Python拥有庞大的社区支持，遇到问题时可以快速找到解决方案。

1.2 安装Python环境 🛠️

安装Python环境是学习Python的第一步，下面我们将详细介绍如何在不同操作系统上安装Python。

Windows系统安装步骤：

  1. 访问Python官方网站：https://www.python.org/downloads/
  2. 点击"Download Python"按钮，下载最新版本的Python安装包。
  3. 运行下载的安装包，在安装界面中勾选"Add Python to PATH"选项。
  4. 点击"Install Now"按钮，等待安装完成。
  5. 安装完成后，打开命令提示符，输入python --version，如果显示Python版本信息，则安装成功。

macOS系统安装步骤：

  1. 访问Python官方网站：https://www.python.org/downloads/
  2. 点击"Download Python"按钮，下载最新版本的Python安装包。
  3. 运行下载的安装包，按照提示完成安装。
  4. 安装完成后，打开终端，输入python3 --version，如果显示Python版本信息，则安装成功。

Linux系统安装步骤：

  1. 大多数Linux发行版已经预装了Python，可以通过在终端中输入python3 --version来检查。
  2. 如果没有安装Python，可以使用包管理器进行安装：
    • Ubuntu/Debian：sudo apt install python3
    • CentOS/RHEL：sudo yum install python3
  3. 安装完成后，在终端中输入python3 --version，如果显示Python版本信息，则安装成功。

1.3 Python开发工具推荐 📝

选择合适的开发工具可以提高编程效率，以下是几个常用的Python开发工具：

  • IDLE：Python自带的集成开发环境，简单易用，适合初学者。
  • PyCharm：功能强大的Python IDE，有社区版（免费）和专业版（付费）。
  • Visual Studio Code：轻量级代码编辑器，安装Python扩展后可以用于Python开发。
  • Jupyter Notebook：交互式笔记本，适合数据分析和学习。

对于初学者，推荐使用IDLE或Visual Studio Code，它们都比较容易上手。

1.4 第一个Python程序 👨‍💻

现在，让我们编写第一个Python程序，体验一下Python的魅力！

使用IDLE编写第一个程序：

  1. 打开IDLE（在Windows上可以在开始菜单中找到，在macOS和Linux上可以在终端中输入idle3）。
  2. 点击"File" → "New File"，创建一个新的Python文件。
  3. 在文件中输入以下代码：
    print("Hello, Python!") print("欢迎来到Python的世界！")
  4. 点击"File" → "Save"，保存文件为hello.py。
  5. 点击"Run" → "Run Module"，运行程序。
  6. 在IDLE的Shell窗口中，你将看到程序输出的内容。

使用命令行运行Python程序：

  1. 创建一个名为hello.py的文件，内容如下：
    print("Hello, Python!") print("欢迎来到Python的世界！")
  2. 打开命令提示符（Windows）或终端（macOS/Linux）。
  3. 导航到保存hello.py的目录。
  4. 输入python hello.py（Windows）或python3 hello.py（macOS/Linux），运行程序。
  5. 在命令行中，你将看到程序输出的内容。

恭喜你！你已经成功编写并运行了第一个Python程序。这只是一个简单的开始，在接下来的课程中，我们将学习更多Python的知识和技能。
